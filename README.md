## About Laravel

CRUD Data Barang with AdminLTE3 Barcode/QRCode

*First step:
- composer install
- rename .env.example to .env (and configure it)
- php artisan key:generate

*Note
- Database : Mariadb / MySQL (qrcode_product)
- Can import excel (.xlsx) with example data 'data-barang.xlsx'
